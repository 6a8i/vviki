Tutorial Page

  • The VViki is a simple web app wiki. Here you can use your imagination to do what you want. But wait! Below there is a little tutorial with some tricks for you. Enjoy it!

  • Double click over the title text to edit it.

  • To print anything from the input box in here is just press shift+enter.

  • .pg for pages and .fdr for folders. This can be change, not defined yet.

  • If you enter a text with multiple paragraphs, the vvike will identify the paragraphs and split it multiple paragraphs into the editor

  • Grab the input box and drop it between some paragraph to add a new paragraph in a especific position
